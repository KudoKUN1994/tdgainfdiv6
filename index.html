Upload Folder Demo

Source code: https://github.com/komputronika/UploadFolder

Pure Javascript, HTML and PHP without any library

This demo allows you to choose a folder via a HTML file input. All files (including nested files) inside that folder are added to the input's files list. This script using webkitdirectory attribute that compatible with browser Chrome, Edge or Firefox.

Choose a folder, and all files in that folder will be uploaded recursively.
